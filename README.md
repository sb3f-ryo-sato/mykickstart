# mykickstart
## 注意事項

1. 以下の設定値は適宜変更すること。

   - rootpw のパスワード
   - user のパスワード
   - PUB_KEY の公開鍵

1. autopart の `--nohome` オプションは CentOS だと無視される模様。

   - [0013969: Anaconda ignores "--nohome" flag to autopart - CentOS Bug Tracker](https://bugs.centos.org/view.php?id=13969)
